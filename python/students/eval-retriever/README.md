# eval-retriever — student version

Student exercise: implement a retrieval quality evaluation harness and log
runs to MLFlow.

**Start by reading [STUDENT_GUIDE.md](STUDENT_GUIDE.md).** It contains the
exercise framing, what you need to implement, hints, and how to verify
your solution.

## Quick commands

```bash
just install-dev     # install deps
just test-metrics    # red right now — your first task is to turn these green
just test            # full suite
just run             # run the CLI end-to-end (requires a running retriever + MLFlow)
```
